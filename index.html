Projects

COVID-19 Global Cases

Visualizing confirmed cases of COVID-19 by country using circles on a map. The area of each circle corresponds to the case of the country it represents. Pan & zoom, and hover over each country for more information.

Line Chart with Multiple Lines

A line chart showing daily count of NYC residents who tested positive for COVID-19. Hover over a point on the chart to display a tooltip showing the case for each borough (sorted in descending order) for the day.

Loftflix

This project is using Laravel, Tailwind CSS and The Movie DB REST API. The feature of this app is the search drop-down. It allows users to search for their favorite movies and get a list of movies from the API.

Loftgram

Loftgram is an Instagram clone application built by an open-source PHP framework Laravel. With user authentication, Loftgram allows users to upload pictures, like a post, write a comment, and follow its users.

Trivia Questions

Built an user interface and single-page application with an open-source JavaScript framework Vue.js. In this project, the app uses Fetch to request a list of questions from the Open Trivia DB API.

My Playlist

My Playlist is playing music with my memories on the phone. The app uses Vue.js reactive features including lifecycle methods, data and methods to create a beginner friendly music app.

Flappy Koopa Troopa
